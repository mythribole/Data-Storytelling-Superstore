Data Storytelling App - Superstore Sales Insights
Project Overview
This project focuses on analyzing the Superstore dataset and presenting meaningful business insights through data storytelling.
The application performs Exploratory Data Analysis (EDA), visualizes important trends, and provides insights and recommendations based on sales and profit performance.

Objectives
•	Understand the structure of the dataset.
•	Perform Exploratory Data Analysis (EDA).
•	Identify sales and profit trends.
•	Analyze customer segments and regions.
•	Generate business insights and recommendations.

Dataset
Superstore Sales Dataset
Dataset Source:
https://raw.githubusercontent.com/leonism/sample-superstore/master/data/superstore.csv

Dataset Introduction
The dataset contains information about:
•	Orders
•	Customers
•	Products
•	Categories
•	Regions
•	Sales
•	Profit
The project analyzes these attributes to understand business performance and support decision-making.

Exploratory Data Analysis
The following analyses were performed:
•	Dataset Shape
•	Dataset Columns
•	First 5 Rows
•	Last 5 Rows
•	Missing Values Analysis
•	Statistical Summary
•	Total Sales
•	Total Profit
•	Total Orders
•	Total Customers

Visualizations
1.	Sales by Region
2.	Profit by Category
3.	Year-wise Sales Trend
4.	Sales by Customer Segment
5.	Top 10 Products by Sales

Key Insights
•	West region generated the highest sales.
•	Technology category contributed significantly to profit.
•	Sales increased steadily over the years.
•	Consumer segment generated the largest share of sales.
•	A small number of products contributed a large portion of revenue.

Recommendations
1.	Focus marketing efforts on high-performing regions.
2.	Increase inventory for top-selling products.
3.	Promote high-profit categories.
4.	Improve performance in low-sales regions.
5.	Use customer segmentation for targeted marketing campaigns.
